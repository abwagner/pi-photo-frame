Connect Dropbox

To enable backups, you need to authorize rclone with your Dropbox account:

  1. Install rclone on a computer with a web browser
  2. Run: rclone authorize "dropbox"
  3. Log in to Dropbox when the browser opens
  4. Copy the JSON token that rclone prints
  5. Paste it below and click "Connect"
